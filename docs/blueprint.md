# **App Name**: PrabhAI

## Core Features:

- Voice Interaction: Enable voice interaction using Whisper API (STT), Eden/OpenRouter API (LLM), and ElevenLabs API (TTS) for natural speech.
- Persona Engine: Allow users to select different personas (e.g., friend, mentor) and store these preferences to tailor AI responses using LLM as a tool.
- Real-Time Knowledge: Provide access to real-time news headlines and articles using the News API. This will use the LLM as a tool to decide when and if to include the latest news snippets in responses.
- Image and Video Generation: Leverage Eden API/OpenRouter API to generate descriptive media content to be used within other AI workflows
- Unforgettable Memory: Implement a persistent memory system that recalls past interactions, preferences, and history for more personalized responses using LLM as a tool.
- Can Be Anything Mode: Allows the assistant to shift between different personas based on user interaction. This mode adapts the personality and voice style based on user input, switching seamlessly between different roles (e.g., friend, girlfriend, mentor, etc.).
- Prabh AI Studio: A tool for creating apps, web games, and much more. Offers a seamless user experience to build and deploy AI-powered projects using intuitive drag-and-drop and code editor features.

## Style Guidelines:

- Primary color: Teal (#008080) for a calming yet engaging feel.
- Secondary color: Light gray (#D3D3D3) for backgrounds and subtle accents.
- Accent: Coral (#FF7F50) for interactive elements and highlights.
- Clean and modern typography for readability.
- Use simple, outline-style icons for a modern look.
- Clean and intuitive layout with a focus on ease of use.
- Subtle transitions and animations to enhance user experience.
